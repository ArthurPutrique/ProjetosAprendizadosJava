# ☕ Projetos Aprendizados Java

Repositório criado para armazenar exercícios, desafios e pequenos projetos desenvolvidos durante meus estudos em Java. O objetivo é praticar os conceitos fundamentais da linguagem e acompanhar minha evolução como desenvolvedor.

## 📚 Sobre o Repositório

Aqui você encontrará diversos projetos desenvolvidos durante cursos, estudos independentes e práticas de programação. Cada pasta representa um exercício ou aplicação focada em um conceito específico da linguagem Java.

## 🚀 Projetos Disponíveis

### 🔢 Comparação de Números

Aplicação simples para comparar valores informados pelo usuário e exibir o resultado da comparação.

### ➖ Número Negativo

Projeto voltado para a prática de estruturas condicionais, verificando se um número é negativo ou não.

### 🧮 Projeto Menu com Cálculos

Sistema de menu interativo que realiza operações matemáticas básicas utilizando entrada de dados pelo terminal.

### ✖️ Tabuada

Programa que gera a tabuada de um número informado pelo usuário, reforçando o uso de laços de repetição.

## 🛠️ Tecnologias Utilizadas

* Java
* IntelliJ IDEA
* Git
* GitHub

## 🎯 Conceitos Praticados

Durante o desenvolvimento destes projetos foram utilizados conceitos como:

* Variáveis e tipos de dados
* Operadores matemáticos e lógicos
* Estruturas condicionais (`if`, `else`)
* Estruturas de repetição (`for`, `while`)
* Entrada de dados com `Scanner`
* Organização de código
* Lógica de programação

## ▶️ Como Executar

1. Clone este repositório:

```bash
git clone https://github.com/ArthurPutrique/ProjetosAprendizadosJava.git
```

2. Abra o projeto desejado em sua IDE Java.

3. Execute o arquivo `Main.java` correspondente ao exercício que deseja testar.
